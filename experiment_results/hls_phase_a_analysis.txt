HLS Phase A Analysis — 2026-04-04
==================================

Experiment: LLM Pragma Prediction on 30 ForgeHLS Kernels
Method: A1 (pragma-only prompt, post-process merge)
Model: qwen2.5-coder:7b (local)

=== KEY RESULTS ===

1. Code Preservation: 30/30 (100%)
   Post-process merge guarantees zero unintended code changes.

2. Pragma Type Match Rate: 98.9% (vs random: 74.4%)
   - PIPELINE identification: 30/30 (100%)
   - Only adpcm had wrong type match (67%)
   - LLM reliably knows WHICH pragma types to use

3. ADRS (distance to Pareto frontier):
   - LLM: 24.84 (skewed by 2 outliers)
   - Worst case: 116.93
   - Median design: 1.44
   - Excluding outliers (AES, wallace_tree): LLM ~0.10

4. Design Percentile:
   - LLM avg: top 15.2% (i.e., better than 84.8% of all designs)
   - Caveat: "nearest design" matching is approximate

=== NOTABLE FINDINGS ===

A. The 7B model ALWAYS correctly identifies PIPELINE (100%).
   This is the most impactful pragma for latency.

B. 13/30 kernels have ADRS < 0.01 — practically on the Pareto frontier.
   These are the "easy" kernels where the LLM's intuition matches HLS best practices.

C. 2 major outliers (AES_Encrypt, wallace_tree_adder) have ADRS > 350.
   AES: 30,468x latency gap, complex encryption with 22 pragmas
   Wallace: 151,251x latency gap, extreme sensitivity to pragma choice

D. fft_strided is the BEST result: top 2.2% of designs (ADRS 0.31)
   The LLM found a near-optimal configuration for FFT.

E. Zero-pragma kernels (syrk, mvt, fdtd-2d, etc.) show ADRS=0.0000
   because LLM added pragmas via for-loop insertion, matching a design
   that already existed in ForgeHLS.

=== LIMITATIONS ===

1. "Nearest design" matching is an approximation. We match LLM pragmas
   to the closest existing ForgeHLS design by pragma overlap count.
   This underestimates LLM quality when it generates novel configurations.

2. No Vivado synthesis — we estimate QoR from ForgeHLS ground truth.
   The LLM might generate valid but untested pragma combos.

3. ADRS is heavily skewed by outliers. Median ADRS = 0.0019 is more
   representative than mean = 24.84.
